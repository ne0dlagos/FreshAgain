# FreshAgain

## 1. Descripción del Proyecto

FreshAgain es una tienda (ficticia) de ropa urbana de segunda mano, que requiere una aplicación móvil nativa para Android que permita a los usuarios registrarse, iniciar sesión, explorar un catálogo de prendas y comprar.

## 2. Integrante

* Daniel Lagos

## 3. Funcionalidades Implementadas a la fecha 26/10/2025

* Arquitectura MVVM.
* Navegación entre pantallas (Registro, Home).
* Formulario de registro de usuario con validaciones.
* Diseño adaptable a diferentes tamaños de pantalla.
* Persistencia local de datos (sesión de usuario) con DataStore.
* Uso de recursos nativos (cámara y galería).
* Animaciones de retroalimentación en formularios.

## 4. Pasos para Ejecutar

1.  Clonar el repositorio: `git clone https://github.com/ne0dlagos/FreshAgain.git`
2.  Abrir el proyecto con Android Studio.
3.  Sincronizar las dependencias de Gradle.
4.  Crear un Dispositivo Virtual con API 33 o superior.
5.  Ejecutar la aplicación.
